# `irys.upload(data, tags)`

Uploads data for [permanent storage](/learn/permanent-storage).

**Parameters**

<div className="-mt-6" />

-   `data`: The data to upload
-   `tags`: Optional [metatags](/developer-docs/tags)

**Returns**

<div className="-mt-6" />- `response`: A JSON object with the following values

```json
response = {
  id, // the transaction id of the upload
  timestamp: // The timestamp of when the transaction was uploaded and verified
}
```

```js
const uploadData = async () => {
	const irys = await getIrys();
	const dataToUpload = "GM world.";
	try {
		const response = await irys.upload(dataToUpload);
		console.log(`Data uploaded ==> https://arweave.net/${response.id}`);
	} catch (e) {
		console.log("Error uploading data ", e);
	}
};
```

<Callout type="info" emoji="ℹ️">
	The transaction id returned as part of the response is used to download the data, simply create a URL with the
	format `https://arweave.net/[transaction-id].`
</Callout>
