# Installing The SDK

The Bundlr SDK can be installed using npm or yarn.

<Tabs items={["npm", "yarn"]}>
	<Tab>npm install @bundlr-network/client</Tab>
	<Tab>yarn add @bundlr-network/client</Tab>
</Tabs>
