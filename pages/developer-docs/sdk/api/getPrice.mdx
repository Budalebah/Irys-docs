# `bundlr.getPrice(numBytes)`

Returns the cost to upload the specified number of bytes.

**Parameters**

<div className="-mt-6" />- `numBytes`: The number of bytes to check the price
for
<div className="-mt-6" />
**Returns**

<div className="-mt-6" />- `price`: Cost to upload `numBytes`. Currency is the
token specified when connecting to the node.Return value is in atomic units, to
convert to standard units use [`bundlr.utils.fromAtomic()`](./utils-fromAtomic)

const numBytes = 1048576; // Number of bytes to check
const priceAtomic = await bundlr.getPrice(numBytes);
