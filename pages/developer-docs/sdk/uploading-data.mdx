# Uploading Data

## `bundlr.upload()`

Uploads any data passed to it, returns a transaction id and a timestamp.

The transaction id returned as part of the response can be used to download the data using an URL with the format `https://arweave.net/[transaction_id]`

```js
// Upload data
const dataToUpload = "Hello world.";
try {
	const response = await bundlr.upload(dataToUpload);
	console.log(`Data uploaded ==> https://arweave.net/${response.id}`);
} catch (e) {
	console.log("Error uploading file ", e);
}
```

## `bundlr.utils.uploadWithReceipt()`

Uploads any data passed to it, returns a cryptographically signed receipt suitable for [Proof Of Provenance](/overview/solutions/proof-of-provenance) applications.

The transaction id returned as part of the response can be used to download the data using an URL with the format `https://arweave.net/[transaction_id]`

```js
// Upload data
const dataToUpload = "Hello world.";
try {
	const receipt = await bundlr.upload(dataToUpload);
	console.log(`Data uploaded ==> https://arweave.net/${receipt.id}`);
} catch (e) {
	console.log("Error uploading file ", e);
}
```
