# `currency(currencyName)`

Queries by currency used for payment. Accepts [any token supported by Irys](/overview/supported-tokens).

<div className="" />
**Parameters**
<div className="-mt-6" />
-   `currencyName`: The name of the [currency](/overview/supported-tokens)

```js
const results = await myQuery
	.search("irys:transactions")
	.currency("solana");
```
