# Downloading data

When you upload data to Irys, our [optimistic cache](/learn/optimistic-cache) ensures your data is immediately available for download.

Data is available for downloading via [gateways](/learn/gateways).

## Downloading Data

The transaction id returned when uploading data is used to generate an URL you can use to download the data.

The format used is `https://arweave.net/[transaction_id]`. For example `https://arweave.net/CO9EpX0lekJEfXUOeXncUmMuG8eEp5WJHXl9U9yZUYA`.
