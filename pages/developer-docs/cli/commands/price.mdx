# `price`

Checks the price to upload the specified number of bytes.

**Required Parameters**

<div className="-mt-6" />

-   `-t`: The [token](/overview/supported-tokens) used for payment.
-   `-h`: URL of the node you will upload to.

## Example

```console
irys price 1000000 -t matic -h https://node1.irys.xyz
```
