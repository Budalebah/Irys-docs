# `price`

Checks the price to upload the specified number of bytes.

**Parameters**

<div className="-mt-6" />

- `-t`: The [token](/overview/supported-tokens) used for payment.
- `-n`: The node you plan to fund (`mainnet` or `devnet`)
- `--provider-url`: RPC URL to use. This is only required when using Devnet.

## Examples

### Mainnet

```console
irys price 1000000 \
  -t matic \
  -n mainnet
```

### Devnet

```console
irys price 1000000 \
  -t matic \
  -n devnet \
  --provider-url https://rpc-mumbai.maticvigil.com
```

<Callout type="info" emoji="ℹ️">
	Devnet RPC URLs change often, use a recent one from https://chainlist.org/
</Callout>
