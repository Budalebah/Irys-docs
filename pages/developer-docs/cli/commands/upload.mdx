# `upload`

Uploads a single file.

**Required Parameters**

<div className="-mt-6" />

-   `-h`: The you'll node upload to
-   `-t`: The [token](/overview/supported-tokens) to use when uploading.
-   `-w`: Your [private key](/developer-docs/cli#using-private-keys).

## Example

```console
irys upload myImage.png -h https://node1.irys.xyz -t matic -w bf20......c9885307
```

## Downloading

Files uploaded via `irys upload` can be downloaded using the transaction ID provided after a successful upload.

```console
irys upload myImage.png -t tagName1 tagValue1 tagName2 tagValue2 -h https://node1.irys.xys -t matic -w bf20......c9885307

Loaded address: 0xac568a981b1370b2e1baa8ce30bd5ac9e28c572d
Uploaded to https://arweave.net/yPX6_rwxfJ_2gTvXSHGoumIvlFR-EwuRlUMmHpeLHUM
```
