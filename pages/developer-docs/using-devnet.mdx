# Using Devnet

Irys' Devnet is for developers who are building projects or learning how to use our SDK. Uploads to Devnet are kept for approximately 60 days. When using Devnet, developers pay using tokens that are available from free public faucets.

To connect to Devnet, supply the address of our Devnet node when instantiating either an `Irys` or `WebIrys` object.

When using Devnet with the `Bundlr` class, you must specify an RPC address. When using the `WebIrys` class, the RPC address is only required when using non-EVM chains.

#### `Bundlr` Class

For Solana:

```js
const url = "https://devnet.irys.xyz";
const providerUrl = "https://api.devnet.solana.com";
const currency = "matic";

const irys = new Irys({
	url, // URL of the node you want to connect to
	currency, // Currency used for payment and signing
	key: process.env.ETH_PRIVATE_KEY, // SOL private key
	config: { providerUrl }, // Optional provider URL, only required when using Devnet
});
```

For Polygon / Matic:

```js
const url = "https://devnet.irys.xyz";
const providerUrl = "https://rpc-mumbai.maticvigil.com;
const currency = "matic";

const irys = new Irys({
	url, // URL of the node you want to connect to
	currency, // Currency used for payment and signing
	key: process.env.ETH_PRIVATE_KEY, // SOL private key
	config: { providerUrl }, // Optional provider URL, only required when using Devnet
});
```

#### `WebIrys` Class And EVM Chains

```js
const bundlr = new WebIrys("https://devnet.bundlr.network", "matic", provider);
```
